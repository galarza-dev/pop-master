Con ./prince megahit se habilitan estas teclas de truco:

Shift + L: pasar al siguiente nivel. 
RetroPie

+ / −: aumentar / reducir el tiempo restante. 
RetroPie

R: “resucitar” (reaparecer donde moriste). 
RetroPie

K: eliminar al guardia en pantalla. 
RetroPie

Shift + S: restaurar un punto de vida (poción pequeña). 
RetroPie

Shift + T: añadir más puntos de vida (poción grande). 
RetroPie

Shift + W: caída lenta (flotar). 
RetroPie

Shift + I: invertir la pantalla (al revés). 
RetroPie

C: mostrar números de la sala actual y adyacentes. 
RetroPie

Shift + C: mostrar números de salas diagonales. 
RetroPie

H / J / U / N: mirar la sala izquierda / derecha / arriba / abajo. 
RetroPie

Shift + B: ocultar/mostrar objetos no animados. 
RetroPie
